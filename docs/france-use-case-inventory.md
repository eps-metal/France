# French E-Invoicing Use Case Inventory

**Source:** `pipeline/a1-source/product/caas/french_einvoicing_use_cases_english_updated.xlsx`
**Prepared:** 2026-03-10
**Primary public source:** https://blob.symtrax.com/doc/fr/tableau-cas-usage.pdf
**Official specs hub:** https://www.impots.gouv.fr/specifications-externes-b2b

## Notes

- Covers use cases 0 (baseline) plus 1–42, including sub-cases 17a/17b, 19a/19b, 22a/22b, and separate rows 20 and 21
- One additional working row for **Interco** (intercompany) — not an official DGFiP numbered use case; included as implementation aid
- Counts vary across reference documents because some count sub-cases differently
- Recommended use: scoping, gap analysis, RFPs, PA/ERP workshops, data-model design
- **Volume shares are not mutually exclusive** — a single invoice can exhibit multiple use cases simultaneously; percentages reflect the share of invoices exhibiting that characteristic, not additive slices
- Volume estimates derived from DGFiP specs, AFNOR XP Z12-014, ASF, INSEE, Symtrax, FNFE-MPE, and sector data (March 2026); no authoritative source publishes use-case-level breakdowns
